## Landing page tutorial project

Project to learn good html css practices

## Dependencies

`vite` for assembly project
`handlebars` for a component approach

## Requirements

nodejs 20.15.0

## Installation

```
npm install
```

```
npm run dev
```

## Preview project

```
npm run preview
```

## Linters

```
npm run stylelint
```

```
npm run lint
```
